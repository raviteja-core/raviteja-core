## 👋 About Me

I'm a developer who likes building **practical, slightly unconventional projects** — from local AI agents to network simulations to small tools that solve a real annoyance. Still figuring out exactly where I want to specialize, and honestly enjoying the exploration.

- 🔭 Currently building **Jarvis** — a local AI agent system
- 🌐 Tinkering with networking & security concepts (DDoS simulation, etc.)
- 🧩 I like projects that are a little weird, a little useful, or both
- 📫 Open to interesting conversations, collabs, and feedback on my code
- ⚡ Fun fact: my repo names are more creative than my variable names
